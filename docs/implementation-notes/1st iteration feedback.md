# Website Discussion

We like the Beta version of website, here’s the link for reference:  
https://motoroverpropage.github.io/motorover.in/beta/

The structure and flow to be inspired from this beta version only, but the Hero Section (above the fold, landing page only) should be taken from the V3 version, attached an image for reference.

After using the V3 version as landing, here are the changes in it:

- The Hero section should not cover the full screen, it has to be a little above the fold, displaying some portion of the next section below it.
- There should be uniformity in the buttons, currently both the buttons (Motorcycle tours, Car tours) have different sizes and colour. The Car tour button is transparent, it is fading in with the background image. For both the buttons, use Yellow button colour and Black text colour, black on yellow is more visible. And both the button sizes should be same.
- At the top right, remove the manual button for changing into night mode. For night mode, we want to keep it automated, from 7 am to 7 pm in light mode and 7 pm to 7 am in night mode. These will change according the the IP and timezone of the visitor on the website. If thats not doable, then it should at least change it with Indian Standard Timezone. But no manual button.
- At the top right corner, remove the Whatsapp icon with full text in it, instead use the Whatsapp icon with just a whatsapp logo, and alongside add icons for Instagram, Facebook and Youtube.
- The logo should be switched with the new one, the new logo is attached to this doc and shared on whatsapp.
- Both the Motorcycle Tours and and Car tours dropdown should have a redirect link at the end saying - View all tours -->

The Flow of website home page should be in this order while scrolling down:

1. Hero Section (attached to it below the stats counter like in Beta version)
2. Motorcycle Tours
3. Self-Drive Road Trips
4. Some Testimonials (with a link, view all testimonials)
5. Why Us
6. Team
7. About Us
8. Some faq (with a link of view all FAQ)

At all the places where images are used in reference to take to another page, we should have the hyperlink on that full section instead of just one button.

## Tour Specific Pages and its changes

- Instead of a Table type Itinerary, we should have a Scrolling type itinerary but not in a table format, with just better flow.
- At the Tour dates section, we should also write these three types of tours:  
  1. Bespoke Tours  
  2. Self-Drive Tours  
  3. Fixed departure guided road trips
- We need a real time currency converter which shows the price in INR when switched, according to the exchange rate of that specific time.
- The tour page sections need to be mobile optimized, currently the sections are going out of the frame.
